Welcome to the JavaScript Unit!

This interactive course will teach you the basics of JavaScript programming through hands-on exercises and practical examples. By the end of these 5 sessions, you'll have built a functional grocery list application while learning core JavaScript concepts.

Each session builds upon the previous one, so we recommend following them in order. You'll have both in-class exercises and resources to continue practicing outside of class.

Start Session 1

What You'll Learn

  • Core JavaScript syntax and concepts
  • DOM manipulation techniques
  • Event handling and user interactions
  • Functions, loops, and control flow
  • Working with arrays and objects
  • Building interactive web applications

Detailed Session Breakdown

Session 1: JavaScript Fundamentals

In this session, you'll learn the basics of JavaScript and how it interacts with the browser. You'll explore variables, data types, and the DOM, and you'll start working with JavaScript in the browser console.

  • Introduction to JavaScript and how it works in the browser
  • Variables, data types, and basic operations
  • Introduction to the DOM (Document Object Model)
  • Using the browser console for JavaScript
  • Selecting and manipulating DOM elements
Start Session 1

Session 2: Functions & DOM Interaction

This session focuses on writing reusable code with functions and parameters. You'll learn how to select and manipulate DOM elements programmatically and handle user events.

  • Functions, parameters, and return values
  • Understanding function scope
  • Working with arrow functions
  • DOM selection and manipulation techniques
  • Event handling and user interaction
Go to Session 2

Session 3: Control Flow & More DOM Manipulation

Learn decision-making with conditional statements and how to work with collections of data using arrays. You'll also learn more advanced DOM manipulation techniques.

  • Conditional statements (if/else)
  • Logical operators for complex conditions
  • Arrays and basic array methods
  • Creating new elements with JavaScript
  • Appending elements to the DOM
Go to Session 3

Session 4: Loops & Advanced DOM Interaction

This session covers repetitive tasks with different types of loops and how to handle dynamic content with event delegation. You'll also learn about objects and form handling.

  • Array forEach method for iteration
  • Using for...of loops for iterables
  • Working with JavaScript objects
  • Object methods and traversing objects
  • Dynamic styling with classList
  • Event delegation for dynamic elements
  • Form data handling techniques
Go to Session 4

Session 5: APIs, JSON & Data-Driven Styling

The final session introduces web APIs, JSON data, and pulls everything together into a practical application. You'll learn to fetch external data and apply dynamic styling based on data values.

  • Understanding web APIs
  • Working with JSON data format
  • Using the Fetch API for HTTP requests
  • Async/await for asynchronous operations
  • Processing and displaying API data
  • Data-driven styling techniques
  • Advanced object techniques
