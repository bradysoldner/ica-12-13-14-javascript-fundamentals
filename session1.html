Introduction to JavaScript

JavaScript is a programming language that allows you to make web pages interactive.

What is JavaScript?

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It was created to make web pages dynamic and interactive.

With JavaScript, you can:

  • Respond to user actions
  • Modify HTML content and CSS styles
  • Validate form data
  • Create animations and visual effects
  • Build complete web applications

Adding JavaScript to a Webpage

There are three main ways to include JavaScript in your HTML:

JavaScript can be written directly in HTML elements using the onclick, onload, and other event attributes.

<button onclick="alert('Hello, World!')">Click Me</button>

Note: This method is generally avoided for more complex code as it mixes HTML and JavaScript.

JavaScript can be placed inside <script> tags in the HTML document:

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello, World!</h1> <!-- Internal JavaScript -->
<script>alert("Hello from JavaScript!");</script> </body> </html>

JavaScript can be placed in external files and linked to the HTML document:

<!-- In your HTML file --> <script src="script.js"></script>
// In script.js alert("Hello from external JavaScript file!");

Benefits of external JavaScript:

  • Separates HTML from JavaScript code
  • Makes both HTML and JavaScript easier to read and maintain
  • Allows the browser to cache the JavaScript file
  • Can be reused across multiple pages

The Document Object Model (DOM)

The DOM is a programming interface for HTML and XML documents that represents the page as a tree of objects.

When a browser loads a webpage, it creates a Document Object Model of the page. The DOM represents the document as nodes and objects, allowing programming languages like JavaScript to interact with the page.

Think of the DOM as a tree-like structure where each branch is a node (HTML element), and the document itself is the root node.

With JavaScript, you can:

  • Find elements in the document
  • Change element content
  • Modify element styles
  • Create new elements
  • Remove existing elements
  • React to user interactions with elements

JavaScript Variables

Variables are containers for storing data values.

In JavaScript, you can declare variables using var, let, or const:

Keyword Description Scope Reassignable Hoisted
var Legacy way to declare variables Function scope Yes Yes
let Modern way to declare variables that can be reassigned Block scope Yes No
const Declares constants that cannot be reassigned Block scope No No
Modern Recommendation: Use const by default, and only use let when you need to reassign a variable. Avoid using var in new code.

Declaring Variables

// Using let (can be reassigned) let name = "John"; name = "Jane"; // This is allowed
// Using const (cannot be reassigned) const pi = 3.14159; // pi = 3.14; // This would cause an error
// Declaring without assigning a value let age; // Value is undefined age = 25; // Assign value later
Variable Naming Rules
  • Names can contain letters, digits, underscores, and dollar signs
  • Names must begin with a letter, $ or _
  • Names are case sensitive (name and Name are different)
  • Reserved words (like JavaScript keywords) cannot be used as names
Examples:
// Valid names let firstName = "John"; let _privateVar = 42; let $price = 9.99;
// Invalid names // let 1stPlace = "Winner"; // let if = true;

JavaScript Data Types

JavaScript has several built-in data types that you can use.

JavaScript is a "dynamically typed" language, which means you don't need to declare the type of a variable when you create it. The type is determined automatically when the program runs.

String

Text values enclosed in quotes

let name = "John"; let message = 'Hello'; let phrase = `${name} says ${message}`;
Number

Integers and floating-point values

let age = 25; let price = 9.99; let negative = -42;
Boolean

Logical values: true or false

let isAdmin = true; let isLoggedIn = false;
Array

Ordered list of values

let colors = ["red", "green", "blue"]; let mixed = [1, "hello", true];
Object

Collection of key-value pairs

let person = { name: "John", age: 25, isStudent: true };
Other Types

Special values in JavaScript

// Undefined - variable declared but not assigned let x; // Value is undefined // Null - intentional absence of value let empty = null; // Symbol & BigInt (newer types)

Checking the Type of a Variable

You can use the typeof operator to check what type a variable holds:

let name = "John"; console.log(typeof name); // "string" let age = 25; console.log(typeof age); // "number" let isAdmin = true; console.log(typeof isAdmin); // "boolean"

The Browser Console

The browser console is a powerful tool for debugging JavaScript and exploring the DOM.

What Can You Do in the Console?

  • Execute JavaScript code directly
  • View output from console.log() statements
  • See errors and warnings
  • Examine variables and objects
  • Interact with the DOM
Common Console Methods
  • console.log() - Output regular messages
  • console.error() - Output error messages
  • console.warn() - Output warning messages
  • console.table() - Display data as a table
  • console.clear() - Clear the console
Try These in Your Console
// Basic output console.log("Hello, World!");
// Math operations console.log(5 + 10);
// Working with variables let name = "JavaScript"; console.log("Hello, " + name + "!");
// Displaying objects let person = { name: "John", age: 25 }; console.log(person);
// Creating a popup alert alert("This is an alert!");

DOM Selection

JavaScript can select and manipulate HTML elements in the DOM.

Before you can modify an element, you need to find and select it. JavaScript provides several methods to select elements from the DOM:

Method Description Returns Example
getElementById() Selects an element by its id attribute Single element document.getElementById("main-heading")
getElementsByClassName() Selects elements by their class HTMLCollection (array-like) document.getElementsByClassName("btn")
getElementsByTagName() Selects elements by their tag name HTMLCollection (array-like) document.getElementsByTagName("p")
querySelector() Selects the first element that matches a CSS selector Single element document.querySelector(".intro")
querySelectorAll() Selects all elements that match a CSS selector NodeList (array-like) document.querySelectorAll("ul li")

Modifying Elements

Once you've selected an element, you can modify its content, attributes, and styles:

// Changing content let heading = document.getElementById("main-heading"); heading.innerHTML = "New Heading Text";
// Changing attributes let link = document.querySelector("a"); link.href = "https://example.com"; link.target = "_blank";
// Changing styles let paragraph = document.querySelector("p"); paragraph.style.color = "blue"; paragraph.style.fontSize = "18px"; paragraph.style.backgroundColor = "#f0f0f0";
Common Element Properties
  • innerHTML - HTML content inside the element
  • textContent - Text content without HTML
  • value - Value of form elements
  • src - Source URL for images
  • href - URL for links
  • style - Access to inline styles
  • className - CSS class(es) of the element
  • id - ID of the element
