Conditional Statements

Making decisions in your code with if, else if, and else statements.

What are Conditional Statements?

Conditional statements allow your program to make decisions based on certain conditions. They execute different code based on whether a condition evaluates to true or false.

JavaScript supports several types of conditional statements:

  • if statement - Executes a block of code if a condition is true
  • if...else statement - Executes one block if a condition is true, another if it's false
  • if...else if...else statement - Tests multiple conditions in sequence
  • switch statement - Selects one of many code blocks to execute

Basic Syntax

// if statement if (condition) { // code to execute if condition is true }
// if...else statement if (condition) { // code to execute if condition is true } else { // code to execute if condition is false }
// if...else if...else statement if (condition1) { // code to execute if condition1 is true } else if (condition2) { // code to execute if condition2 is true } else { // code to execute if all conditions are false }

Logical Operators

Combining and manipulating boolean values with AND, OR, and NOT operators.

Types of Logical Operators
  • AND (&&) - Returns true if both operands are true
  • OR (||) - Returns true if at least one operand is true
  • NOT (!) - Returns the opposite of the operand's boolean value

These operators are essential for creating complex conditions in your programs.

// AND operator if (age >= 18 && hasLicense) { console.log("You can drive"); }
// OR operator if (isWeekend || isHoliday) { console.log("No work today!"); }
// NOT operator if (!isLoggedIn) { console.log("Please log in first"); }

Arrays

Working with ordered collections of data.

What are Arrays?

Arrays are special variables that can hold multiple values. They store data in an ordered list, where each item has a numeric index starting from 0.

Creating Arrays
// Array literal syntax const fruits = ["apple", "banana", "orange"];
// Array with multiple data types const mixedArray = [42, "hello", true];
Accessing Array Elements
const colors = ["red", "green", "blue"];
// Access by index console.log(colors[0]); // "red" console.log(colors[2]); // "blue"
// Modify by index colors[1] = "yellow"; console.log(colors); // ["red", "yellow", "blue"]

Array Methods Optional

Powerful tools for working with arrays: push, pop, map, filter, and more.

Method Description Example
push() Adds one or more elements to the end of an array fruits.push("kiwi");
pop() Removes the last element from an array fruits.pop();
unshift() Adds one or more elements to the beginning of an array fruits.unshift("strawberry");
shift() Removes the first element from an array fruits.shift();
splice() Changes array content by removing/replacing elements fruits.splice(1, 1, "grape");
slice() Returns a shallow copy of a portion of an array const newArray = fruits.slice(0, 2);
indexOf() Returns the first index at which a given element is found fruits.indexOf("banana");
join() Joins all elements of an array into a string fruits.join(", ");

Advanced Array Methods Optional

Modern JavaScript provides powerful higher-order array methods for transforming and filtering data:

const numbers = [1, 2, 3, 4, 5];
// map() - creates a new array by calling a function on each element const doubled = numbers.map(num => num * 2); // doubled is [2, 4, 6, 8, 10]
// filter() - creates a new array with elements that pass a test const evenNumbers = numbers.filter(num => num % 2 === 0); // evenNumbers is [2, 4]
// reduce() - reduces array to a single value const sum = numbers.reduce((total, num) => total + num, 0); // sum is 15

Creating HTML Elements

Dynamically generating HTML elements with JavaScript.

Creating Elements

JavaScript allows you to create HTML elements programmatically using the document.createElement() method.

The basic process for adding new elements to a page is:

  1. Create the element using document.createElement()
  2. Set properties (text content, attributes, classes, etc.)
  3. Append the element to an existing element in the DOM
// Create a new paragraph element const paragraph = document.createElement("p");
// Set its text content paragraph.textContent = "This is a dynamically created paragraph.";
// Add a CSS class paragraph.className = "dynamic-text";
// Create a container reference const container = document.getElementById("container");
// Append the paragraph to the container container.appendChild(paragraph);

Modifying the DOM Tree

Adding, updating, and removing elements from the document.

Common DOM Modification Methods

Once we've created elements, we can use various methods to add, remove, or replace them in the DOM:

Method Description
appendChild() Adds a node to the end of the list of children of a specified parent node
insertBefore() Inserts a node before a reference node as a child of a specified parent node
removeChild() Removes a child node from the DOM and returns the removed node
replaceChild() Replaces a child node with a new node
// Create a new element const newElement = document.createElement("li"); newElement.textContent = "New Item";
// Append to an existing element parentElement.appendChild(newElement);
// Insert before a specific element parentElement.insertBefore(newElement, referenceElement);
// Remove an element parentElement.removeChild(childElement);
// Replace an element parentElement.replaceChild(newElement, oldElement);
